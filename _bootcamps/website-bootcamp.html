Detailed Curriculum

This coding bootcamp is structured into 7 phases. Following are the key phases in this bootcamp.


Phase Goal Session(s)
1 Preparation 1
2 HTML Basics 2

4 CSS Intermediate 2

{% include util/note.html note="1 Session = (1 - 1.5 Hrs), 2 Sessions = 1 Week, Total = 16 Sessions = 8 Weeks = 2 Months." class="w-100 mb-0" %}

Phase 1 - Preparation

This phase helps prepare students mentally and set up the development environment as well.

It includes:

  • Introduction of Instructor and Students
  • Brief of how the bootcamp will work
  • Introduction to the communication platforms
  • Web, HTTP and REST Introduction - How the web works
  • Introduction the necessary tools required for this bootcamp
  • Git/GitHub workflow and basic commands
  • Software Development workflow in an IT company

Phase 2 - HTML Basics

This phase helps prepare students with prerequisite concepts of HTML5 needed for the understanding of Website development.

  • Understanding Basic Layout and Hello World
  • Understanding Element, Tag and Attribute
  • Using Meta Tag
  • Using Headings and Paragraph
  • Using Div and Span
  • Performing Text Formatting in HTML
  • Hyperlink in HTML
  • Using Image in HTML
  • Inline and Block elements in HTML
  • Creating list in HTML
  • Creating table in HTML
  • HTML Form and Input Elements
  • ID and Class in HTML
  • Semantic Tags

Phase 3 - CSS Basics

In this phase, students will understand the basics of CSS required to style the Website.

  • CSS introduction, types and comment
  • Introduction to browser developer tools
  • Using CSS selector
  • CSS colors and its types
  • Fonts in CSS
  • Enhancing with Google Fonts
  • CSS Units
  • CSS Backgrounds
  • CSS Borders
  • Margin, Padding and Box Model

Phase 4 - CSS Intermediate

In this phase, students will learn more about CSS and try to understand concepts like float, auto margin, style the navigation bar, CSS positioning etc.

  • CSS rounding corners
  • Text formatting in CSS
  • Max-width and margin auto
  • Float and Clear
  • Challenge: Create a basic image gallery
  • Links, its states and button styling
  • CSS display: inline, block and inline-block
  • Styling navigation menu bar
  • Styling a login form
  • Challenge: Style a sign-up form
  • Positioning with CSS
  • CSS Z-index property
  • CSS Specificity and !important Rule

Phase 5 - JavaScript Basics, DOM and BOM

This phase will help students understand the JavaScript basics, document object model (DOM) and browser object model (BOM).

  • Introduction and first JavaScript(JS) code
  • JS statements, syntax and comments
  • Variables: var, let, and const
  • Operators and Assignemnt
  • Data Types in JavaScript
  • Functions in JavaScript
  • Objects in JavaScript
  • Events in JavaScript
  • String and its methods
  • Array and its methods
  • Conditional statements in JavaScript
  • Loops in JavaScript
  • Switch in JavaScript
  • Comparision and Logical Operators
  • typeof operator and type conversion
  • JavaScript Hoisting concept and strict mode
  • Concept of 'this' keyword
  • Arrow function in JavaScript
  • Introduction to Class and Module
  • DOM Concepts
    • Introduction to DOM and document object
    • Accessing HTML elements using DOM
    • Updating HTML content with DOM
    • Updating Styling with DOM
    • Working with Event and Event listener with DOM
    • DOM Node and Navigatio
  • BOM Concepts
    • Introduction to BOM and window object
    • Working with different popup
    • Working with window's screen object
    • Working with window's location object
    • Working with window's history object
    • Working with setTimeout and setInterval timing functions
    • Introduction to JavaScript cookie

Phase 6 - jQuery and Bootstrap v5

In this module, we will learn about the most popular JavaScript framework called jQuery and its common usage.
After that, then, we will learn about the usage of the most popular CSS framework called Bootstrap.

  • jQuery Concepts
    • jQuery Introduction and getting started
    • jQuery Syntaxes
    • Selectors in jQuery
    • Working with events in jQuery
    • Important methods in jQuery
  • Bootstrap v5 Concepts
    • Bootstrap Introduction and getting started
    • Container, Grid System and Responsiveness concepts
    • Typography and colors
    • Working with images
    • Working with buttons
    • Working with Cards
    • Dropdown and Collapse component
    • Working with Carousel and Modal
    • Flexbox in Bootstrap
    • Working with Forms
    • Working with Basic Templates

Phase 7 - Projects building

In this phase, students will learn how to build a complete Website project from scratch, understand the project building workflow, comprehend and find path through the issues, understand the complexities of building website etc.

Student will build following website in this phase:

  • Portfolio Website
  • School or College Website
  • Restaurant Website
  • Consultancy Website

Phase 8 - Git, GitHub and Deployment

In this phase, students will learn basics of source control management with Git and GitHub.
Here, they will learn how to manage code with Git and publish in GitHub so that it will be easily accessible with others for collaboration.
Finally, students will learn how to deploy the website in the platform like Netlify.

  • Getting started with Git and GitHub
  • Initializing website project with Git
  • Git basic commands
  • Publish the website to GitHub
  • Getting started with Netlify
  • Deploy the website in Netlify
